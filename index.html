История EcmaScript

  1. 1995, Brendan Eich создал JavaScript за 10 дней;
  2. 1997, ECMA-262 версия 1 (ES1);
  3. 1998 ECMA-262 версия 2: редакционные правки;
  4. 1999, ECMA-262 версия 3;
  5. 2007, ECMA-262 версия 4: отклонена (предложения сохранены в проекте ECMAScript Harmony);
  6. 2009, ECMA-262 версия 5;
  7. 2011, ECMA-262 версия 5.1
  8. 2015, ECMA-262 версия 6 (ES6, ES2015, Harmony);
  9. 2016, ECMA-262 версия 7 (ES7)

основы

Объявления переменных: let, const

let — правильный var:

  • Область видимости переменной блок, а не функция;
  • Существует только после объявления, а не undefined по умолчанию;
  • Создаётся на каждую итерацию цикла

const — настоящая константа

  • нельзя изменить после присвоения

Строки: шаблоны, Юникод

Новый вид кавычек для строк:

  • разрешён перевод строки
  • вставка выражений при помощи ${…}
  • своя функция шаблонизации

Юникод:

  • правильная работа с 4-х байтными символами (например, иероглифы)
    • codePointAt()
    • fromCodePoint()

Строки: шаблоны, Юникод

Добавлены методы:

  • repeat(count): повторение count раз.
  • startsWith(searchString[, position]): проверка — начинается ли с searchString
  • includes(searchString[, position]): поиск внутри searchString
  • endsWith(searchString[, position]): проверка — заканчивается ли с searchString
